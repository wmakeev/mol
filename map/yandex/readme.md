# $mol_map_yandex

Simple [Yandex Maps](https://tech.yandex.ru/maps/doc/jsapi/2.1/) wrapper.

## [Online demo](http://eigenmethod.github.io/mol/#demo=mol_map_yandex)

## Usage example

```
<= Spb $mol_map_yandex
	center /
		60
		30
	zoom 10
```

## Properties

**`zoom( next? : number ) : number`**

Zoom level.

**`ccenter( next? : [ number , number ] ) : [ number , number ]`**

Position of center.
